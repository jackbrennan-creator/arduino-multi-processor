DataFlow Library 1.0.1
----------------------

The DataFlow library support programs written for the Arduino Multi-Processor.

Documentation
-------------

See https://carinasagittarius.co.uk/articles/MSP2/Arduino_Multi-Processor.html

This version has been update to use pin change interrupts to allow a sender to monitor the "received" response from a receiver.

Installation
------------

To install this library, just copy the entire unZIPed DataFlow folder into your Arduino libraries folder or use the "Sketch.Include Library.Add ZIP Library" option in the Arduino IDE to install the downloaded ZIP file directly.
